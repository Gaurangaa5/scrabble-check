# Scrabble Word Checker

A simple offline Scrabble word checker made with HTML, CSS and JavaScript.

## Files

Keep these files in the same folder:

- `index.html`
- `words.txt`
- `CSW24.txt`
- `Cairo-Regular.ttf`
- `Cairo-Bold.ttf`

## How to Use

1. Download this repository as a ZIP.
2. Extract the ZIP.
3. Open `index.html` in your web browser.
4. Click **Choose File** and select either `words.txt` or `CSW24.txt`.
5. Type a word and press **Check Word** (or press **Enter**).

## Note

The **GitHub Pages** version is **not fully offline**. It works online, but you'll need an internet connection to access the website.

To use the checker completely **offline**, download the repository as a ZIP, extract it, and open `index.html` directly from your computer.
